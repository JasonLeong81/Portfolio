Data Science

Expression Recognition using Supervised ML algorithms

This is a project in which the dataset contains multiple class label such as happy, sad, anxious, confused, and so on. There are also different types of sadness such as very sad or just a little sad. The goal is to correctly predict what expression is presented in images. The input data was huge therefore I applied dimensionality reduction algorithms such as principal component analysis and t-SNE. I trained several classification and regression algorithms. For classification, I applied linear SVM, kernel SVM, and decision trees with entropy as purity measurement. For regression, I applied linear regression and lasso regression.
